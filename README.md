## Psychological Counseling Soul Healing Communication Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/48bda9c75f4446dcbe9d7c581276f0fa/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, psychological counselors, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. System Announcements
5. Psychological Counselor Appointment, Rate Psychological Counselors, View Qualification Certificates
6. Mental Health Column
7. Stress Testing
8. Small Notes
9. Personal Center, Personal Information Modification
10. Chat
11. Change Password
12. My Test Data
13. My Counselor Appointments
14. Post Small Notes, View, Modify My Published Small Notes

**Psychological Counselor Functions:**
1. Login
2. Registration
3. Change Password
4. Modify Personal Information
5. Counselor Appointment Management
6. Chat Management


**Administrator Functions:**
1. Login
2. Personal Center
3. System Announcement Management
4. Psychological Counselor Management
5. Mental Health Column Management
6. Stress Testing Management
7. Test Data Management
8. Counselor Appointment Management
9. Counselor Appointments
10. Small Notes Management
11. Carousel Management
12. Chat Management
